## Examples

### Generate responses using Cohere's command model


This example demonstrates how to use the Cohere command processor to generate responses for a record's `.Payload.After` field.
The processor sends the input text from the configured "request.body" to the Cohere API and stores the model's response into the configured "response.body"

#### Configuration parameters

<Tabs groupId="config-params">
  <TabItem value="yaml" label="YAML">
```yaml
version: 2.2
pipelines:
  - id: example
    status: running
    connectors:
      # define source and destination ...
    processors:
      - id: example
        plugin: "cohere.command"
        settings:
          apiKey: "apikey"
          backoffRetry.count: "0"
          backoffRetry.factor: "2"
          backoffRetry.max: "5s"
          backoffRetry.min: "100ms"
          model: "command"
          prompt: "hello"
          request.body: ".Payload.After"
          response.body: ".Payload.After"
```
