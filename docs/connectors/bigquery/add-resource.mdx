This guide will walkthrough adding a BigQuery resource to Meroxa. Before adding, see [BigQuery Setup](/docs/destinations/bigquery/setup).

## Credentials

To add a BigQuery resource, you will need the following credentials:

- GCP Project ID - A [Google Cloud Project ID](https://support.google.com/googleapi/answer/7014113?hl=en).
- BigQuery Dataset - A BigQuery dataset (see [Setup](/docs/destinations/bigquery/setup)).
- Service Account JSON - A valid Service Account credentials with access to BigQuery (see [Setup](/docs/destinations/bigquery/setup)).

## Adding Resource
<Tabs
  defaultValue="cli"
  values={[
    {label: 'CLI', value: 'cli'},

<TabItem value="dashboard">

1. Navigate to the [Add Resource](https://dashboard.meroxa.io/resources/new) page.

![Add BigQuery Resource to Meroxa Catalog](/images/docs/destinations/bigquery/add-bigquery.png)

2. Configure access credentials:

- Project ID - Google Cloud Project ID
- BigQuery Dataset - A BigQuery dataset (see [Setup](/docs/destinations/bigquery/setup)).
- Service Account JSON - Valid Service Account credentials with access to BigQuery (see [Setup](/docs/destinations/bigquery/setup)).

3. Select "Add".

</TabItem>
<TabItem value="cli">

Using the [CLI](/cli/installation-guide), you can run the following command to add a new resource:

```shell
meroxa resource add my-bigquery \
--type bigquery \
--url bigquery://$GCP_PROJECT_ID/$GCP_DATASET \
--client-key "{ $GCP_SERVICE_ACCOUNT_JSON }"
```

In the command above, replace the following variables with valid credentials from your BigQuery environment:

- `GCP_PROJECT_ID` - Google Cloud Project ID
- `$GCP_DATASET` -  A BigQuery dataset (see [Setup](/docs/destinations/bigquery/setup)).
- `$GCP_SERVICE_ACCOUNT_JSON` - Valid Service Account credentials with access to BigQuery (see [Setup](/docs/destinations/bigquery/setup)).

</TabItem>
</Tabs>
 

You can view your resource with [`meroxa resource ls`](/cli/cmd/meroxa-resources-list) or in the [Dashboard](https://dashboard.meroxa.io/resources).
