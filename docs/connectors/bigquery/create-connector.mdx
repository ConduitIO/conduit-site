The BigQuery Destination Connector allows you to load data from a pipeline into a BigQuery Table.

Before creating a destination connector:

1. [Setup](/docs/destinations/bigquery/setup) - Setup your BigQuery instance and acquire the credentials needed to talk to Meroxa.
2. [Add Resource](/docs/destinations/bigquery/add-resource) - Add a BigQuery resource to your Meroxa Resource.
3. Create a new [pipeline](/docs/pipelines/overview#creating-pipelines) and add a [source connector](/docs/sources/overview).

Using the [CLI](/cli/installation-guide), you can run the following command to add a new destination to a pipeline:

```shell
meroxa connector create to-bigquery-connector \
 --to my-bigquery \
 --input $STREAM_NAME \
 --pipeline my-pipeline
```

The command above creates a new destination connector called `to-bigquery-connector`, sets the destination to a resource named `my-bigquery`, and configures the input with a [stream](/docs/pipelines/streams).
