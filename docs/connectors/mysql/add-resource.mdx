This guide will walkthrough adding a MySQL resource to Meroxa. Before adding, see [MySQL Setup](/docs/sources/mysql/setup/overview).

## Credentials

To add a MySQL resource, you will need an **Access URL**:

The Access URL contains the credentials needed to access your instance:

```
mysql://$MYSQL_USER:$MYSQL_PASS@$MYSQL_URL:$MYSQL_PORT/$MYSQL_DB
```

In the example above, replace following variables with valid credentials from your Postgres environment:

- `$MYSQL_USER` - MySQL Username
- `$MYSQL_PASS` - MySQL Password
- `$MYSQL_URL` - MySQL URL
- `$MYSQL_DB` - MySQL Database Name
- `$MYSQL_PORT` - MySQL Port (e.g., 5432).

<TabItem value="cli">

Using the [CLI](/cli/installation-guide), you can run the following command to add a new resource:

```shell
meroxa resource add my-mysql \
--type mysql \
--url mysql://$MYSQL_USER:$MYSQL_PASS@$MYSQL_URL:$MYSQL_PORT/$MYSQL_DB \
```
