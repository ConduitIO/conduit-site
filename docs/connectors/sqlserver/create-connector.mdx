The SQL Server Destination Connector allows you to load data from a pipeline into a SQL Server Table.

Before creating a destination connector:

1. [Setup](/docs/destinations/sqlserver/setup/overview) - Setup your SQL Server instance and acquire the credentials needed to talk to Meroxa.
2. [Add Resource](/docs/destinations/sqlserver/add-resource) - Add a SQL Server resource to your Meroxa Resource.

## Input

The input of SQL Server is a table name.

Using the [CLI](/cli/installation-guide), you can run the following command to add a new destination to a pipeline:

```shell
meroxa connector create to-sqlserver-connector \
 --to my-sqlserver \
 --input $STREAM_NAME \
 --pipeline my-pipeline
```

The command above creates a new destination connector called `to-sqlserver-connector`, sets the destination to a resource named `my-sqlserver`, and configures the input with a [stream](/docs/pipelines/streams).
