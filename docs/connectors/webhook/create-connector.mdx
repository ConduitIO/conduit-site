The Webhook Destination Connector allows you to `POST` data from a pipeline to an `HTTP` endpoint.

Before creating a destination connector:

1. [Add Resource](/docs/destinations/webhook/add-resource) - Add a Webhook resource to your Meroxa Resource.
2. Create a new [pipeline](/docs/pipelines/overview#creating-pipelines) and add a [source connector](/docs/sources/overview).

Then, to configure Webhook as a destination:

Using the [CLI](/cli/installation-guide), you can run the following command to add a new destination to a pipeline:

```shell
meroxa connector create to-webhook-connector \
 --to my-webhook \
 --input $STREAM_NAME \
 --pipeline my-pipeline
```

The command above creates a new destination connector called `to-webhook-connector`, sets the destination to a resource named `my-webhook`, and configures the input with a [stream](/docs/pipelines/streams).
