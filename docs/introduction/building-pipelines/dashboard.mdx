The Meroxa Dashboard is a web-based interface is designed to make building pipelines simple and easy to build. You may build pipelines using a drag-and-drop interface.

![Meroxa Dashboard](/images/docs/introduction/building-pipelines/dashboard.png)

## Getting Started

To get started, navigate to the [Meroxa Dashboard](https://dashboard.meroxa.io/onboarding/source?onboardingStep=source). There you will be presented with the steps to get your pipeline up and running.

![Meroxa Dashboard Getting Started](/images/docs/introduction/building-pipelines/dashboard-getting-started.png)
