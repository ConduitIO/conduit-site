## Starting Conduit

To get started, you need to [install and start](/docs/introduction/getting-started) Conduit. You may [even add Conduit to your `$PATH`](/guides/adding-a-connector-with-rest-swagger-ui).

```bash
./conduit
```

## Open Swagger UI

To open the Swagger UI, open your browser and navigate to `http://localhost:8080/openapi`. 

PHOTO

This UI allows you to interact with the API and create connectors. It also serves as a reference for the API.
