We're excited to announce the release of one of our most useful connectors, the [Conduit HTTP Connector](/docs/using/connectors/list/http), as version [0.3.0](https://github.com/conduitio-labs/conduit-connector-http/releases/tag/v0.3.0). 


<!--truncate-->

### Key Highlights

This update incorporates the latest [Conduit Connector SDK](https://github.com/ConduitIO/conduit-connector-sdk/releases/tag/v0.13.4), making it possible to add as a [built-in connector to your Conduit build](/docs/using/connectors/additional-built-in-plugins).
