[Conduit DynamoDB Connector v0.3.1](https://github.com/conduitio-labs/conduit-connector-dynamodb/releases/tag/v0.3.1) is
now available, introducing a new feature for AWS authentication.

<!--truncate-->

### Key Highlights

- [**Make AWS credentials optional with fallback to default credential chain**](https://github.com/conduitio-labs/conduit-connector-dynamodb/pull/164).
This change allows the connector to authenticate with AWS using either static credentials or AWS default config chain,
providing more flexibility in how the connector accesses DynamoDB resources. Thanks [@annsway](https://github.com/annsway) for the contribution!

If you have any questions, or more feedback feel free to join our [Discord Community](https://discord.meroxa.com/).

:::tip
For more information, check out the [DynamoDB Connector documentation](/docs/using/connectors/list/dynamodb).
:::
