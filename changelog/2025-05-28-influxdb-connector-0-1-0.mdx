Today, we're adding a new Connector to our extensive [list of Connectors](/docs/using/connectors/list). [Conduit InfluxDB Connector v0.1.0](https://github.com/conduitio-labs/conduit-connector-influxdb/releases/tag/v0.1.0) is now available, and ready to use as both source and destination connector.

<!--truncate-->

If you're using [InfluxDB](https://www.influxdata.com/), you're ready to use it with Conduit as a [standalone connector](/docs/core-concepts#standalone-connector), or as part of a custom [Conduit Build](/docs/developing/connectors/using-a-custom-connector).

:::tip
For detailed setup and configuration instructions, refer to our [documentation](/docs/using/connectors/list/influxdb).
:::
