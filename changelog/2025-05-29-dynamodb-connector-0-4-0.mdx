[Conduit DynamoDB Connector v0.4.0](https://github.com/conduitio-labs/conduit-connector-dynamodb/releases/tag/v0.4.0) is
now available, introducing the **DynamoDB Destination Connector**!

<!--truncate-->

### Key Highlights

- [**DynamoDB Destination Connector Added**](https://github.com/conduitio-labs/conduit-connector-dynamodb/pull/150)
This release brings a destination connector to complement the existing DynamoDB source. It supports all standard
operations: `snapshot`, `insert`, `update`, and `delete`, making it easier to stream data into DynamoDB tables from
other sources.
